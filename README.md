#  The Fiasco – Restaurant Website

##  Description

Ce projet consiste en l’intégration d’une page web moderne pour un restaurant fictif nommé **The Fiasco**, à partir d’une maquette graphique.

L’objectif principal était de :

* Reproduire fidèlement le design fourni
* Compléter les sections manquantes
* Rendre le site **entièrement responsive** (mobile, tablette, desktop)
* Améliorer la structure HTML/CSS

---

##  Fonctionnalités

*  Header avec navigation + menu responsive (burger)
*  Hero section avec image de fond et texte centré
*  Sections “Gastronomy” et “Recipes”
*  Menu dynamique avec cartes de plats
*  Section Chef
*  Formulaire de réservation
*  Section blog / recettes
*  Section localisation
*  Footer complet

---

##  Technologies utilisées

* **HTML5**
* **CSS3 (Flexbox + Grid)**
* Responsive Design (Media Queries)
* Google Fonts / Fonts locales

---

##  Responsive Design

Le site est entièrement adapté pour :

*  Mobile
*  Tablette
*  Desktop

Améliorations apportées :

* Suppression des tailles fixes (ex: height: 5400px)
* Utilisation de layouts flexibles
* Ajout de media queries
* Navigation mobile avec menu burger

##  Problèmes corrigés

*  Layout rigide avec grid globale sur le body
*  Hauteur fixe empêchant le responsive
*  Sections vides ou incomplètes
*  Mauvaise organisation CSS
*  Mauvaise gestion mobile

---

##  Objectifs pédagogiques

Ce projet permet de pratiquer :

* L’intégration web à partir d’une maquette
* Le responsive design
* L’organisation d’un projet front-end
* L’optimisation CSS

##  Aperçu

Design basé sur une maquette de restaurant moderne avec une identité visuelle élégante (noir + doré).
